# 클로드 추출기 (Claude Exporter) v1.3.4

Chrome 확장 프로그램 본체입니다. 저장소 전체 안내는 [상위 README](../README.md)를 참고하세요.

## 설치

`chrome://extensions/` → 개발자 모드 → **압축해제된 확장 프로그램을 로드** → **이 폴더** 선택.

## 사용

Claude 대화 페이지 → 확장 아이콘 → **MD / TXT / PDF** 선택.

## 저장 규칙 요약

- **파일명:** `Claude-{제목10자,띄어쓰기없음}-{YYMMDD}.md`
- **「새 채팅」:** 첫 사용자 메시지를 이어 붙여 10자
- **파일 맨 위 제목:** 저장 파일명과 확장자 포함 (예: `# Claude-파일명규칙개선해줘-260525.md`)
